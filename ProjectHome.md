# _Half Byte Loader BETA_ #

This software allows you to execute home-made (_homebrews_) applications made for PSP on both PSP and PS Vita. It's basically a pure user-mode firmware. _Homebrews_ requiring kernel privileges will not execute properly.

**It does NOT run ISO/CSO**
